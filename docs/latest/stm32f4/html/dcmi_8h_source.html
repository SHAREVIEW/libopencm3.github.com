111 /**
112  * DCMI masked interrupt status register
113  *
114  * This DCMI_MIS register is a read-only register. When read, it returns the current masked
115  * status value (depending on the value in DCMI_IER) of the corresponding interrupt. A bit in
116  * this register is set if the corresponding enable bit in DCMI_IER is set and the corresponding
117  * bit in DCMI_RIS is set.
118  */
